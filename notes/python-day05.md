> Python sets __name__ variable value for all modules. If the file is run the variable sets as "main", otherwise module's actual name. Hence, this allows same file to be directly runnable and also be reusable module. Basically if __name__ == __main__ block serves as the main entry point if the module is directly run
> Difference between script and module -> a file when imported and used in another file is called module, and the same file when run on its own is called script
> Package -> folder-based grouping of modules. __init.py__ file is used to initialize the folder as package
> Json objects are read as Python dictionary
> Traceback: Detailed error message printed on exception occurence
